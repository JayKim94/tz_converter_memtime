#### Installation

1. Install dependencies
```sh
npm install
```

2. Run local server
```sh
npm run dev
```

#### Test

1. Run Vitest
```sh
npm test
```

2. Open Vitest UI
```sh
http://localhost:51204/__vitest__/
```
